### 描述

更新当前用户语言信息

### 输入参数

| 参数名称        | 参数类型   | 必选 | 描述                         |
|-------------|--------|----|----------------------------|
| language    | string | 是  | 语言类型（中文为 `zh-cn`，英文为 `en`） |

### 请求示例

```json5
// 请求体
{
    "language": "zh-cn"
}
```

### 状态码 200 的响应示例

```json5
{
    "data": {
        "language": "zh-cn"
    }
}
```

### 响应参数说明

| 参数名称     | 参数类型   | 描述                         |
|----------|--------|----------------------------|
| language | string | 语言类型（中文为 `zh-cn`，英文为 `en`） |
