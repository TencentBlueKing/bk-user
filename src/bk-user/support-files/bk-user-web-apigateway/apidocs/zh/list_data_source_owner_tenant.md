### 描述

获取本租户所有数据来源的租户信息（包括协同租户）

### 状态码 200 的响应示例

```json5
{
  "data": [
    {
      "id": "default",
      "name": "默认租户"
    },
    {
      "id": "collab_tenant_1",
      "name": "协同租户1"
    },
    {
      "id": "collab_tenant_2",
      "name": "协同租户2"
    }
  ]
}
```

### 响应参数说明

| 参数名称                | 参数类型   | 描述       |
|---------------------|--------|----------|
| id                  | string | 租户 ID    |
| name                | string | 租户名称     |
