### 描述

（分页）查询部门间关系列表

### 输入参数

| 参数名称      | 参数类型 | 必选 | 描述                |
|-----------|------|----|-------------------|
| page      | int  | 否  | 页码，从 1 开始         |
| page_size | int  | 否  | 每页数量，默认 10，最大 500 |

### 请求示例

```
// URL Query 参数
page=1&page_size=5
```

### 状态码 200 的响应示例

```json5
{
    "data": {
        "count": 2,
        "results": [
            {
                "id": 1,
                "parent_id": null,
            },
            {
                "id": 2,
                "parent_id": 1,
            }
        ]
    }
}
```

### 响应参数说明

| 参数名称          | 参数类型 | 描述                      |
|---------------|------|-------------------------|
| department_id | int  | 部门唯一标识                  |
| parent_id     | int  | 父部门 ID（若为 null 则表示为根部门） |
