### 描述

批量查询租户公共属性

### 输入参数

| 参数名称    | 参数类型   | 必选 | 描述                                        |
|---------|--------|----|-------------------------------------------|
| lookups | string | 是  | 精确查询的属性名，多个以逗号分隔，限制数量为 100，每个值最大输入长度为 255 |


### 请求示例

```
// URL Query 参数
lookups=key_1,key_2
```

### 状态码 200 的响应示例

```json5
{
    "data": [
        {
            "key": "key_1",
            "value": "value_1",
        },
        {
            "key": "key_2",
            "value": "value_2"
        }
    ]
}
```

### 响应参数说明

| 参数名称  | 参数类型   | 描述     |
|-------|--------|--------|
| key   | string | 公共属性名称 |
| value | string | 公共属性值  |
