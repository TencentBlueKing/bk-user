### 描述

（分页）查询部门列表

### 输入参数

| 参数名称      | 参数类型 | 必选 | 描述          |
|-----------|------|----|-------------|
| page      | int  | 否  | 页码，从 1 开始   |
| page_size | int  | 否  | 每页数量，默认为 10 |

### 请求示例

```
// URL Query 参数
page=2&page_size=2
```

### 状态码 200 的响应示例

```json5
{
  "data": {
    "count": 2,
    "results": [
      {
        "id": 3,
        "name": "部门B",
        "parent_id": 1,
      },
      {
        "id": 4,
        "name": "中心AA",
        "parent_id": 2,
      }
    ],
  }
}
```

### 响应参数说明

| 参数名称      | 参数类型   | 描述     |
|-----------|--------|--------|
| id        | int    | 部门唯一标识 |
| name      | string | 部门名称   |
| parent_id | int    | 父部门 ID |
